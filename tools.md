##### 本地服务器文件列表H5AI

- docker 安装 H5AI

  1. 安装docker

  2. $ docker pull clue/h5ai

  3. $ docker run -d -p 80:80 -v PWD :/var/www clue/h5ai

  4. 浏览器 输入 localhost查看

     > PWD:  /Users/用户名/h5ai   /h5ai: 新建文件夹用于存放需要共享的文件

参考: [<https://hub.docker.com/r/clue/h5ai/>](<https://hub.docker.com/r/clue/h5ai/>)
